# 🏙️ Civic AI – Smart Complaint Analyzer

## 📌 Project Overview

Civic AI is an intelligent complaint analysis system developed as part of **Decode Labs Project 3**. The system helps citizens report civic issues and automatically identifies the responsible government department, predicts the priority level, and provides recommended actions.

The goal is to use AI-inspired decision-making to improve the handling of public complaints and support smart city initiatives.

---

## 🎯 Problem Statement

Design an AI-based recommendation system that:

* Accepts user inputs (preferences, requests, or problems)
* Analyzes patterns using logical rules or similarity techniques
* Generates meaningful recommendations or decisions

This project applies those concepts to civic issue management by recommending the appropriate department and actions based on a user's complaint.

---

## 💡 Solution

Civic AI analyzes complaint descriptions submitted by users and:

1. Understands the complaint context
2. Identifies the relevant department
3. Determines the urgency level
4. Recommends actions for citizens
5. Recommends actions for government authorities

---

## ⚙️ Features

### 🏢 Department Classification

Automatically assigns complaints to departments such as:

* Electricity
* Roads & Infrastructure
* Water Supply
* Sanitation
* Traffic Management
* Public Health
* Police & Public Safety

### ⚠️ Priority Prediction

Classifies complaints into:

* High Priority
* Medium Priority
* Low Priority

### 👤 Citizen Guidance

Provides safety recommendations and immediate actions for citizens.

### 🏛️ Government Recommendations

Suggests actions that the responsible department should perform.

### 📊 Interactive Dashboard

* Complaint analytics
* Department-wise statistics
* Priority distribution
* Visual charts and reports

### 🕒 Complaint History

* Stores complaint records
* Enables future analysis
* Tracks system usage

---

## 🔄 Workflow

1. User enters a complaint.
2. Civic AI processes the complaint text.
3. The system identifies the responsible department.
4. Priority level is predicted.
5. Recommendations are generated.
6. Results are displayed through an interactive dashboard.
